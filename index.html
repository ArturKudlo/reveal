Classes in JS ES2015 classes


class People {
    constructor(lastname) {
        this.lastname = lastname;
        this.age = 0;
    }

    set newAge(value) {
        this.age = value;
    }

    get represent() {
        return `My name is ${this.name}. I am ${this.age} years old.`;
    }
}
						

const artur = new People('Artur');
artur.represent // My name is Artur. I am 0 years old.
artur.newAge = 18;
artur.represent // My name is Artur. I am 18 years old.

Inheritance

Classical (JAVA, C++)

  • classes inherit from classes
  • are being created subclasses : hierarchical classes

Prototypal(JavaScript)

  • A prototype is a working object instance
  • objects are inherited from other objects

Inheritance ES2015


class PeopleMan extends People {
    constructor(lastname) {
        super(lastname);
    }

    say(message) {
        console.log('man');
        super.say(message);
        console.log('man');
    }
}
						

	const ivanov = new PeopleMan('Ivanov');
	ivanov.say('hello, I'm Ivanov');
	// man
	// Ivanov: 'hello, I'm Ivanov'
	// man
